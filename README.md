# **유니티로 구현한 LAV-AD**

### 조작키

- 이동: WASD
- 발사: 좌클릭
- 확대: 우클릭 (1인칭 한정)
- 1인칭, 3인칭 전환: V
- 차량 상태 리셋: R

### 특징

- 탄퍼짐과 탄피배출 구현
- 카메라 심도 효과
- 서스펜션 구현
- 후진 시 후미등 점등
- 재질에 따라 달라지는 총알 피격 이펙트
- 전진 및 후진시 차량의 뒷바퀴 2쌍에 흙먼지 이펙트 생성
- 발사가 오래 지속되면 포신 과열
